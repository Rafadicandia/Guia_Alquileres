# Mi Guía Alquiler

Una aplicación web construida con React y Vite para gestionar y explorar información sobre alquileres de propiedades.

## 🚀 Características

- Interfaz moderna y responsive diseñada con Tailwind CSS
- Componentes React reutilizables
- Desarrollo rápido con Vite
- Estilos con Tailwind CSS v4

## 🛠️ Tecnologías

- **Frontend**: React 19
- **Build Tool**: Vite 6
- **Estilos**: Tailwind CSS 4
- **Linting**: ESLint

## 📋 Requisitos Previos

Antes de ejecutar el proyecto, asegúrate de tener instalado:

- Node.js (versión 18 o superior)
- npm o yarn

## ⚡ Instalación

1. Clona el repositorio:
```bash
git clone <url-del-repositorio>
```

2. Navega al directorio del proyecto:
```bash
cd Guia_Alquileres
```

3. Instala las dependencias:
```bash
npm install
```

## ▶️ Ejecución

### Modo desarrollo

```bash
npm run start
```

Esto abrirá la aplicación en `http://localhost:5173`

### Build de producción

```bash
npm run build
```

### Preview del build

```bash
npm run preview
```
